# [Document] Scaffold panel指令

> [注意] 仅Windows可用

## 1 指令介绍

唤起由Easygui编写的简易的桌面端仪表盘。

## 2 demo

![123](https://i.loli.net/2021/07/11/d6SnOlWjsTxHqQb.gif)

## 3 使用说明

在`main.py`所在目录下运行以下编译指令

``` python
python main.py panel
```

或根据具体情况运行，既确保运行的python版本 `py >= 3.6`

```python
python3 main.py panel
```
